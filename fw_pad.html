アプリの使用方法と注意事項（必ずお読みください）

基本的な使い方

  1. 調査中にメモを入力して「この地点を記録する」をタップします。
  2. 地図に記録ピンが即時追加され、リストにデータが追加されます。
  3. 入力中のメモや記録データは、端末のブラウザに自動保存されます。オフラインやブラウザ終了時もデータは消失しません。
  4. 「CSV保存」や「JSON保存」をタップすると、ファイルとして端末にダウンロードできます。

スマホで実機のGPSを使うための注意

スマートフォンのセキュリティ制限により、GPS（位置情報取得）は「https://」から始まる安全なウェブサイト上でしか動作しません。

  • このHTMLファイルをそのままスマホに転送して開いた場合（URLが file:/// で始まる場合）、GPS機能はブロックされます。
  • スマホで外に持ち出して使う場合は、このHTMLファイルを GitHub Pages や Netlify Drop などの無料ホスティングサービスにアップロードし、そのURLにアクセスしてご利用ください。
  • ※擬似モードでの地図タップによる座標指定や、データの保存・出力機能はローカル環境のままでも完全に動作します。

保存件数と安全な運用について

データはブラウザの「ローカルストレージ」に保存されます。容量的には約5,000〜10,000件以上保存可能ですが、スマホの動作が重くなるのを防ぐため、1回の調査につき「500〜1,000件」を目安としてください。

💡 おすすめの安全な運用ルーティン

  1. 調査が終わるごとに「CSV保存」 または 「JSON保存」 で端末にエクスポートする。
  2. ダウンロードしたファイルをクラウドやPC等にバックアップする。
  3. バックアップの確認後、アプリ内の「全消去」ボタンを押してリストを空っぽにし、翌日の調査に備える。
